# Flower Shop Menu – Java Console Application

A simple interactive console program for a flower shop. Users can order bouquets, calculate prices based on flower type, color, and size, store orders (up to 10), and view summary statistics.

## Features
- Interactive menu-driven interface
- Choose from 12 flower types, 7 colors, and 3 sizes
- Automatic price calculation (flower + color) × size multiplier
- Input validation and error handling (InputMismatchException)
- Stores order details in memory
- Displays statistics: min/max price, range, total orders, revenue, average price

## Technologies
- Java (JDK 8 or higher)
- Scanner for console input
